Kom igång med Claude Code på tre sätt

Claude Code är samma verktyg oavsett hur du öppnar det. Här är tre vägar in, från enklast till mest kraftfull: i Claude-appen, i VS Code och i terminalen. Allt steg för steg, för Windows och Mac. Välj det sätt som passar dig, du behöver bara ett.

Innan du börjar: Claude Code kräver en betald Claude-plan (Pro, Max, Team eller Enterprise). Gratisplanen ger inte tillgång. Terminalen och VS Code funkar även med ett Claude Console-konto (betala per användning).

Vilket sätt ska du välja?

Tre vägar till samma verktyg. Tryck på ett kort för att hoppa dit.
Sätt 1

Claude-appen

En vanlig app på datorn med knappar och fönster. Inget krångel i terminalen.

Bäst för nybörjare
 Sätt 2

VS Code

Claude Code som panel direkt i din kodeditor, bredvid filerna du jobbar i.

Bäst om du redan kodar
 Sätt 3

Terminalen

Kommandoraden. Mest kraftfull, alla kommandon, och går att skripta.

Bäst för full kontroll
Sätt 1

I Claude-appen

Den enklaste vägen. En app på datorn med chatt, kodvy och knappar. Inget i terminalen.
1

Ladda ner appen

Gå till nedladdningssidan och hämta appen för ditt system. Sidan känner oftast av om du har Windows eller Mac.

Ladda ner Claude-appen
2

Installera

Windows

Kör filen som laddades ner (.exe) och följ stegen. Starta sen Claude från Startmenyn.

Mac

Öppna .dmg-filen och dra Claude till mappen Program. Öppna Claude därifrån.

3

Logga in

Logga in med ditt Claude-konto. Du behöver en betald plan (Pro, Max, Team eller Enterprise). Ser du "Upgrade required" betyder det att planen inte räcker.

4

Öppna fliken Code

Längst upp finns tre flikar: Chat, Cowork och Code. Tryck på Code.

5

Välj din mapp och ställ första frågan

Tryck på Select folder och välj mappen för ditt projekt. Skriv sen i rutan:

Första prompten
vad gör det här projektet?

Claude läser dina filer och sammanfattar projektet. Tips: skriv / i rutan för att se alla kommandon.

Sätt 2

I VS Code

Claude Code som en panel direkt i kodeditorn VS Code, bredvid filerna du jobbar i.
1

Installera VS Code (om du inte redan har det)

Ladda ner och installera VS Code för Windows eller Mac.

Ladda ner VS Code
Krav: VS Code version 1.98 eller nyare. Tillägget behöver inte Node.js eller terminal-versionen installerad först.
2

Installera tillägget "Claude Code"

I VS Code, tryck Ctrl+Shift+X (Mac: Cmd+Shift+X) för att öppna Extensions. Sök på Claude Code och välj det från Anthropic, tryck Install. Du kan också installera direkt från Marketplace:

Öppna tillägget i Marketplace
3

Logga in

Öppna en fil i VS Code, så dyker en stjärnikon (Spark) upp uppe till höger. Tryck på den, välj Sign in och logga in i webbläsaren som öppnas.

4

Öppna ditt projekt

Gå till File och Open Folder, välj din projektmapp. Öppna en fil och tryck på stjärnikonen, så öppnas Claude Code-panelen till höger.

5

Ställ din första fråga

Skriv i panelen:

Första prompten
förklara vad den här filen gör

Claude ser filen du har öppen automatiskt. Skriv @filnamn för att peka på andra filer, och / för att se kommandon.

Bra att veta: tillägget har egen Claude Code inbyggd. Vill du köra kommandot claude i VS Codes terminal behöver du även installera terminal-versionen (Sätt 3).
Sätt 3

I terminalen

Kommandoraden. Mest kraftfull, alla kommandon, och går att skripta. Du behöver inte Node.js, Claude Code är ett eget program.
1

Installera Claude Code

Anthropic rekommenderar installeraren nedan, den uppdaterar sig själv. Kör raden som matchar ditt system.

Windows (PowerShell)

PowerShell
irm https://claude.ai/install.ps1 | iex

Mac (och Linux)

Terminal
curl -fsSL https://claude.ai/install.sh | bash
Föredrar du en pakethanterare? Windows: winget install Anthropic.ClaudeCode. Mac: brew install --cask claude-code. Har du Node.js: npm install -g @anthropic-ai/claude-code. De uppdaterar inte automatiskt.
2

Kontrollera att det funkar

Öppna en ny terminal och kör:

Terminal
claude --version

Du ska få ett versionsnummer. Står det "command not found", stäng och öppna terminalen igen så att den hittar programmet.

3

Logga in

Kör claude. Första gången öppnas webbläsaren där du loggar in. Sen är du inloggad och slipper göra om det.

Terminal
claude
4

Starta i ditt projekt

Gå till mappen för projektet och starta Claude där:

Terminal
cd sokvag/till/ditt/projekt
claude

Skriv sen din fråga, till exempel vad gör det här projektet?

5

Kommandon att börja med

/init skapar en CLAUDE.md med regler för projektet. /help visar allt. Shift+Tab slår på planläget (Claude planerar innan den ändrar). /clear rensar chatten. /exit avslutar.
